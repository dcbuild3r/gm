# GM Odysseas

A script to setup a Linux or MacOS system with:
- GUI & CLI apps
- Development toolchain for Ethereum, Rust and Javascript
- Dotfiles-

## Instructions
```shell
# help
./gm.sh -h
```
![](https://i.imgur.com/msg8vCr.png)
```shell
# Install everything
./gm.sh -a
```
